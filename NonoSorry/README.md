# Bootstrap Website

This project is a simple website built using Bootstrap. It features a background image and two interactive buttons.

## Project Structure

```
bootstrap-website
├── public
│   ├── index.html        # Main HTML document
│   ├── css
│   │   └── styles.css    # Custom CSS styles
│   ├── js
│   │   └── main.js       # JavaScript functionality
│   └── assets
│       └── images        # Directory for images
├── .gitignore            # Files to ignore by Git
├── package.json          # npm configuration file
└── README.md             # Project documentation
```

## Features

- **Responsive Design**: Utilizes Bootstrap for a responsive layout.
- **Background Image**: The website has a background image set to `D:\NonoSorry\img\nono1.jpg`.
- **Interactive Buttons**: Includes two buttons labeled "和好" and "不要".

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd bootstrap-website
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Open `public/index.html` in a web browser to view the website.

## Usage

- Click on the "和好" button to perform the desired action.
- Click on the "不要" button to perform an alternative action.
